# UI/UX Pro Max References

## Industry Reasoning Rules
Each industry has a specific "personality":
- **Fintech/Banking**: Trust & Authority. Avoid harsh animations or "AI purple" gradients.
- **Gaming**: Immersive & High Contrast. Use glowing effects and sharp edges.
- **E-commerce**: Clean & Action-oriented. High visibility for CTAs.

## Design System Template (MASTER.md)
When generating a design system, define:
- **Colors**: Primary, Secondary, CTA, Background, Text.
- **Typography**: Heading font, Body font, Weights.
- **Effects**: Shadow depths (sm, md, lg), Blur levels for glassmorphism.
- **Grid**: Gap sizes, Padding standards.

## UX Checklist
- [ ] Responsive breakpoints defined?
- [ ] Contrast ratios compliant?
- [ ] Interactive states (hover, active, disabled) clear?
- [ ] Consistent spacing (using a 4px or 8px base)?
