# UI/UX Pro Max

This skill provides "design intelligence" for building professional user interfaces, automating the creation of comprehensive design systems based on industry standards.

## Core Design Workflow

When a UI/UX task is requested, follow this structured process:

1. **Requirement Analysis**: Identify the product type, industry, and specific user requirements.
2. **Design System Generation**: Establish patterns, styles, colors, typography, and effects before implementation.
3. **Validation**: Check against UX guidelines and accessibility standards (WCAG AA).

## Design Standards

### Visuals & Layout
- **Responsiveness**: Design for 375px, 768px, 1024px, 1440px.
- **Components**: Prefer Bento-style cards for modular layouts.
- **Interactions**: 
  - `cursor-pointer` on all clickable elements.
  - Transitions should be 150-300ms.
- **Accessibility**: Minimum text contrast of 4.5:1.

### Typography & Icons
- **Fonts**: Use specific pairings (e.g., Cormorant Garamond / Montserrat).
- **Icons**: Use SVG icons (Heroicons/Lucide). Do NOT use emojis as icons.

### MTier Specifics (Integrated)
- **Theme**: Orange OLED / Glassmorphism.
- **Minecraft Icons**: Use `mc-icon` class for pixelated rendering.
- **Badges**: Use `animate-float` and `animate-pulse-glow` for important elements.
