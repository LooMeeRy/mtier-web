# AI Context & Documentation

This folder contains files specifically designed for AI agents (like Gemini, Claude, or GitHub Copilot) to understand the project's current state, instructions, and architecture.

## Files in this directory:
- **`PROJECT_STATUS.md`**: The source of truth for current progress, completed tasks, and pending features. **AI must update this file after every significant change.**
- **`AGENTS.md`**: (If present) Specific instructions for different agent personas.
- **`CLAUDE.md`**: (If present) Project-specific rules and patterns for AI agents.

## Instructions for AI Agents:
1. **Read First**: Always read `PROJECT_STATUS.md` before starting work to understand where we left off.
2. **Update Always**: Every time you complete a task or change the project structure, update the activity log and task list in `PROJECT_STATUS.md`.
3. **Consistency**: Follow the established tech stack and UI patterns described in the documentation.
